# MSS-XCADE Control Library

A library for controlling the Iowa Scaled Engineering MSS-XCADE (Block Signal Custom) modular signal system board within the Arduino environment.  The Block Signal Custom is a platform for building complex model railroad signal interlocking plants that are compatible with the open Modular Signal System standard.

For full developer details, please see the [Developer's Guide](https://docs.iascaled.com/SimpleSig/Block%20Signal%20Custom/developer_guide/)
